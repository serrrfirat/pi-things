# Build in Public: Claude Code Skills Showcase

Create engaging Twitter threads that showcase your Claude Code skills setup, demonstrate what each skill actually DOES, and tease Synapse auto-generation.

## Context

You have 12+ Claude Code skills:
- **product-manager-toolkit** - RICE prioritization, PRD templates, customer interview analysis
- **front-end-designer** - Production-grade UI with distinctive design (not generic AI aesthetics)
- **marketing-specialist** - Demand gen playbooks, CAC calculator, HubSpot integration
- **authentication** - JWT, OAuth2, session management, RBAC patterns
- **document-writer** - Documentation in Metabase's conversational style
- **interviewer** - Deep-dive questions using AskUserQuestion tool
- **twitter-thread** - Viral thread frameworks with hooks and CTAs
- **revenuecat** - In-app subscription implementation
- **react-native-app** - Cross-platform mobile with navigation, state, native modules
- **screenpipe-integration** - SDK integration and screen capture patterns
- **skill-writer** - Meta-skill for creating new skills
- **domain-name-researcher** - Domain brainstorming with availability checks

**Synapse**: Your tool that auto-generates Claude Code skills by analyzing screen activity patterns.

## Thread Formula for Skills Showcase

```
1/ [HOOK - Numbers + curiosity]
   "I have 12 Claude Code skills that..."
   "Everyone's talking about Claude Code skills. Here's my setup:"

2/ [Social proof / Context]
   Reference @thekitze's 29 skills, @nummanali's GitHub Actions
   Position yourself in the conversation

3-7/ [2-3 CONCRETE EXAMPLES]
   Don't just list names - show what they DO
   Format: Skill name → Actual capability → Outcome

8/ [The collection overview]
   Quick hits of remaining skills

9/ [THE TEASE - Synapse]
   "But here's the thing..."
   "What if these wrote themselves?"
   Hint at auto-generation

10/ [CTA]
   Follow for more, link to Synapse, ask for favorites
```

## Writing the Concrete Examples

This is the most important part. Transform skill names into impact stories:

### BAD (just listing):
```
3/ I have:
- product-manager-toolkit
- front-end-designer
- authentication
```

### GOOD (showing value):
```
3/ product-manager-toolkit

I say "prioritize these features" and it runs RICE scoring automatically.

Customer interview → structured insights in seconds.

PRD templates that actually match how I think.
```

```
4/ front-end-designer

Most AI-generated UI looks... AI-generated.

This skill produces distinctive designs that don't scream "I used Claude."

It's opinionated. It has taste.
```

```
5/ authentication

"Add login to this app"

It doesn't ask which method. It implements JWT + refresh tokens + session management.

Because it knows my patterns.
```

## The Synapse Tease

Build intrigue without over-explaining:

```
9/ But here's what I'm really excited about...

What if these skills wrote themselves?

I'm building something that watches my workflows and auto-generates Claude Code skills.

Patterns that fire together → automations that save hours.

More soon. 👀
```

Or more subtle:
```
9/ Plot twist:

I didn't write most of these manually.

I built a tool that watches what I do repeatedly...

...and generates the skills automatically.

It's called Synapse. Thread coming soon.
```

## Quality Checklist

Before delivering:

- [ ] Hook creates immediate curiosity
- [ ] 2-3 skills have CONCRETE examples (not just names)
- [ ] Examples show outcomes, not features
- [ ] Community references included (@thekitze, @nummanali, etc.)
- [ ] Synapse tease builds intrigue without over-explaining
- [ ] CTA is specific and actionable
- [ ] Each tweet under 280 characters
- [ ] Thread flows naturally
- [ ] Would make someone want to build their own skills
